Problema

Rediseño de página web para la venta de accesorios para telefonía celular, y además ofrecer servicio técnico para celulales de marca “Apple”.

Objetivo y solución

Rediseñar la página web para facilitar el uso con el fin de insitar la compra online de productos y que el usuario realice consultas por este medio. Para su solución se investigó como es la navegabilidad de los usuarios en páginas e-commerce, para poder completar un proceso de compra de manera intuitiva.

Investigación

Se realizó un Benchmarking de empresas que se localizan en la misma área geográfica que Tecnoboss. Se tomó una competencia directa y dos indirectas para realizar esta parte de la investigación. Se comparó la navegación, vocabulario y diseño para observar las fortalezas y debilidades de cada una para incorporar al diseño de nuestra página web.A partir de los resultados de la investigación, se realizaron entrevistas virtuales se pudo crear los USERPERSONAS, se idearon 2 tipos de usuarios, uno que desea comprar accesorios electrónicos y otro que necesita del servicio técnico.

Se realiza un card sorting para diferenciar las secciones y facilitar la navegabilidad de los usuarios, logrando idear el USERFLOW

Finalmente se realizaron pruebas de usabilidad y con ello los wireframes, se crearon.

Conclusión

El proceso implicó un rediseño integral del sitio web de comercio electrónico, comenzando con un análisis de la página web existente y no funcional. Posteriormente, se llevó a cabo una investigación para recopilar datos comparativos que permitieran un examen exhaustivo de los competidores en las proximidades de la ubicación geográfica del establecimiento "Tecnoboss". Después de evaluar las fortalezas y debilidades de ambos competidores y el estado actual del sitio web comercial, se estableció un flujo de usuarios para animar los wireframes y representar visualmente la navegabilidad del usuario.

A continuación, se realizaron pruebas de usabilidad iterativas que culminaron con la creación de un kit de interfaz de usuario para dar vida al rediseño del sitio web.

Objetivo y solución

Informar y enseñar a los usuarios a crear o adquirir cremas y cosméticos naturales con el fin de evitar el uso de componentes negativos para el ecosistema. Por lo que como solución se diseñó una aplicación que enseñe a los usuarios a crear o adquirir cosméticos naturales teniendo en cuenta las características médicas de cada uno, como tipo de piel, alergias, entre otras.

Investigación

Los plásticos que se encuentran en la superficie de los oceanos representan solo un 15% del total. Por otra parte están los microplásticos, que estan presente en el 79% de los artículos realizados con plásticos, en ellos podemos detectar los productos de dermocosmética. De estos envases solo el 9% es reciclado. A partir de esta premisa, comenzamos a determinar quienes serían nuestros Userpersonas para ello se realizaron entrevistas virtuales, que determinaron con 2 tipos de usuarios, uno que realiza produtos de cosmética natural y la otra que los compra
